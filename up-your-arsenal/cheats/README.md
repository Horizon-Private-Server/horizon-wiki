## Freecam
Only works in local play. NTSC only. Will crash online. Instructions from Agent Moose:
```
Controls:
L1 + R1 + L3: Activate
L1 + R1 + R3: Deactivate
Joysticks: Move
Dpad: Move/Strafe
Hold L1: Double Speed
Hold R1: Half Speed
Square: Occlusion On/Off
```
